RECOMMENDED REQUIREMENTS:

 WebServer with PHP support.
 MySQL 8.0.44 and above.     ( https://dev.mysql.com/downloads/mysql/ )
 PHP 8.1.33 and above.        ( https://www.php.net/downloads.php)

 For windows users:
  If you are using Apache2.4,  ( https://httpd.apache.org/ )
   we recommend PHP VS16/VS17 x64 Thread Safe.

 Opera Web Browser 10.
 Java SE 5.0.


INSTALLATION:

 Copy MiniManager into any of your webserver folders.

 Configuration:

 - Recommended
    Copy /scripts/config.dist.php as /scripts/config.php,
    Make changes there.

 Import from /SQL/*.sql to Database set in $mmfpm_db from /script/config.php


If some aspect of installation settings are not covered by this guide,
 check FAQ.txt and our forums for further details.
